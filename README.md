## Installation

1. Install client with yarn

```bash
  cd client
  yarn install
```

2. Install server with yarn

```bash
  cd server
  yarn install
```

3. Prepare env file for server
```bash
  # .env
  OPENAI_API_KEY=[your key]
```

4. Start server
```bash
  yarn start
```
